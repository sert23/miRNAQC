mirnaQC can work with 3 types of input and you can provide one or several samples:

  • Upload a reads file (typically fastq or fastq.gz) E.g: IK_exo.fastq.gz
  • Provide a link to a reads file (typically fastq or fastq.gz) E.g: https://arn.ugr.es/srnatoolbox/static/IK_exosomes.fastq.gz
  • Provide an accession for a SRA run (they start with SRR,ERR or DRR e.g. SRR1563062)
